Naming conventions
==================

Naming conventions are also referred as:

- Naming style
- Case convention

----------------

| ----------------- | ------------- | ------------- | ------------- | ------------- |
| `_` (underscore)  | Snake case    | Ada case      | Macro case    | -             |
| `-` (dash)        | Kebab case    | Train case    | Cobol case    | -             |
| `.` (dot)         | Dot notation  | -             | -             | -             |
